### 02. JS的HelloWorld

1. JS的代码需要编写到`script`标签中

```html
<html>
    <head>
        <script>
            // 弹出一条警示框
            alert("Hello World");

            // document.write()可以向body输出一个内容
            document.write("This is a word.");

            // 向控制台输出一个内容
            console.log("This is a log.");
        </script>
    </head>
</html>
```
